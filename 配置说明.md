# 配置说明

## 📝 环境变量配置

### 1. 文件位置

环境变量文件：`.env.local`（已创建）

### 2. 获取 Contentful 配置

#### 步骤 1：注册 Contentful

1. 访问：https://www.contentful.com/
2. 点击 "Get started for free"
3. 使用邮箱注册账号

#### 步骤 2：创建空间（Space）

1. 登录后，点击 "Create space"
2. 选择 "Free" 计划
3. 输入空间名称（如：公司官网）
4. 点击 "Create space"

#### 步骤 3：获取 API Key

1. 在左侧菜单点击 **Settings** → **API keys**
2. 点击 **Content delivery / preview tokens** 标签
3. 你会看到：
   - **Space ID**：复制这个值
   - **Content delivery API - access token**：复制这个值

### 3. 配置 .env.local 文件

打开 `.env.local` 文件，填入你的配置：

```env
# Contentful 配置
CONTENTFUL_SPACE_ID=你的-space-id（从 Contentful 复制）
CONTENTFUL_ACCESS_TOKEN=你的-access-token（从 Contentful 复制）

# 网站配置
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_SITE_NAME=你的公司名称
```

**示例**：

```env
CONTENTFUL_SPACE_ID=abc123xyz456
CONTENTFUL_ACCESS_TOKEN=def789ghi012jkl345mno678pqr901stu234
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_SITE_NAME=我的公司
```

### 4. 保存文件

保存 `.env.local` 文件后，重新启动开发服务器。

## ✅ 验证配置

配置完成后，运行：

```bash
npm run dev
```

如果看到 "Ready" 消息，说明配置成功！

如果出现错误，检查：
1. Space ID 和 Access Token 是否正确
2. `.env.local` 文件是否在项目根目录
3. 文件格式是否正确（没有多余的空格）
