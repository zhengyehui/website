## 📋 5 分钟快速开始

### 1. 注册 Contentful（2分钟）

1. 访问：https://www.contentful.com/
2. 点击 "Get started for free"
3. 注册账号（使用邮箱即可）
4. 创建新空间（Space）
5. 获取 API Key：
   - 点击左侧 Settings → API keys
   - 复制 **Space ID** 和 **Content delivery API - access token**

### 2. 配置项目（1分钟）

```bash
cd D:\project\website

# 复制环境变量文件
copy env.local.example .env.local

# 或者使用 PowerShell
Copy-Item env.local.example .env.local
```

然后编辑 `.env.local`，填入你的 Contentful 信息：

```env
CONTENTFUL_SPACE_ID=你的-space-id
CONTENTFUL_ACCESS_TOKEN=你的-access-token
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_SITE_NAME=你的公司名称
```

### 3. 安装并运行（2分钟）

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

打开浏览器访问：http://localhost:3000

## 🚀 部署到 Vercel（免费，5分钟）

### 方法一：通过 GitHub（最简单）

1. 在 GitHub 创建新仓库
2. 推送代码：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/website.git
   git push -u origin master
   ```

3. 访问 https://vercel.com/
4. 点击 "Add New Project"
5. 导入你的 GitHub 仓库
6. 添加环境变量：
   - `CONTENTFUL_SPACE_ID`
   - `CONTENTFUL_ACCESS_TOKEN`
7. 点击 "Deploy"

完成！你的网站会自动部署，并获得一个免费域名（如：your-company.vercel.app）

### 方法二：通过 Vercel CLI

```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login

# 部署
vercel

# 添加环境变量
vercel env add CONTENTFUL_SPACE_ID
vercel env add CONTENTFUL_ACCESS_TOKEN

# 重新部署
vercel --prod
```

## 📝 内容更新（非技术人员也能操作）

1. 登录 Contentful 后台
2. 点击 "Content" → "Add entry"
3. 编辑内容
4. 点击 "Publish"
5. 网站自动更新（Vercel 会自动重新构建）

## 🎨 自定义网站

### 修改公司名称

编辑以下文件，搜索 "公司名称" 并替换：
- `components/Header.tsx`
- `components/Footer.tsx`

### 修改颜色

编辑 `tailwind.config.js`，修改 `primary` 颜色值

### 修改联系方式

编辑 `components/Footer.tsx` 和 `app/contact/page.tsx`

## 📧 配置表单提交

### 使用 Formspree（推荐，免费）

1. 访问 https://formspree.io/ 注册
2. 创建新表单
3. 复制表单 ID
4. 编辑 `app/contact/page.tsx`
5. 找到注释的 Formspree 代码，取消注释
6. 替换 `YOUR_FORM_ID` 为你的表单 ID

1. **安装失败**：确保 Node.js 版本 >= 18
2. **Contentful 连接失败**：检查 `.env.local` 中的配置
3. **部署失败**：检查 Vercel 中的环境变量是否正确
