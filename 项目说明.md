## 🚀 立即开始（3步）

### 第1步：注册 Contentful（2分钟）

1. 访问 https://www.contentful.com/
2. 注册免费账号
3. 创建空间（Space）
4. 获取 API Key（Settings → API keys）

### 第2步：配置项目（1分钟）

```bash
cd D:\project\company-website

# 复制环境变量文件
copy env.local.example .env.local
```

编辑 `.env.local`，填入你的 Contentful 信息

### 第3步：启动（1分钟）

```bash
# 安装依赖
npm install

# 启动
启动.bat
```

或手动：

```bash
npm run dev
```

访问：http://localhost:3000
