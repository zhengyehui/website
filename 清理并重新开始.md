# 清理并重新开始 Git 仓库

## 🎯 如果你想完全重新开始

### 步骤 1：删除 Git 历史

```powershell
cd D:\project\website

# 删除 .git 文件夹
Remove-Item -Recurse -Force .git
```

### 步骤 2：重新初始化

```bash
# 初始化新的 Git 仓库
git init

# 添加所有文件
git add .

# 创建第一次提交
git commit -m "Initial commit"
```

### 步骤 3：重新连接远程仓库

```bash
# 添加远程仓库
git remote add origin https://github.com/zhengyehui/website.git

# 强制推送（覆盖远程历史）
git push -u origin main --force
```

---

## 🚀 一键脚本

运行 `删除所有文件.bat`，选择选项 2。

---

## ⚠️ 重要提示

- `--force` 会覆盖远程仓库的历史
- 确保没有其他人在使用这个仓库
- 删除前备份重要文件

---

## ✅ 完成后

你的仓库将是一个全新的、干净的 Git 仓库，只包含当前的文件。
