## 🚀 完整操作流程

### 第一步：注册并配置 Contentful（5分钟）

#### 1.1 注册 Contentful

1. 访问：https://www.contentful.com/
2. 点击右上角 "Get started for free"
3. 使用邮箱注册账号（Google/GitHub 登录也可以）
4. 验证邮箱（如果需要）

#### 1.2 创建空间（Space）

1. 登录后，点击 "Create space"
2. 选择 **"Free"** 计划（完全免费）
3. 输入空间名称（如：公司官网）
4. 点击 "Create space"

#### 1.3 获取 API Key

1. 在左侧菜单点击 **Settings**（设置）
2. 点击 **API keys**（API 密钥）
3. 点击 **Content delivery / preview tokens** 标签
4. 你会看到两个重要信息：
   - **Space ID**：类似 `abc123xyz456`（复制这个）
   - **Content delivery API - access token**：类似 `def789ghi012...`（复制这个）

#### 1.4 配置 .env.local 文件

1. 打开 `D:\project\website\.env.local` 文件
2. 将刚才复制的值填入：

```env
CONTENTFUL_SPACE_ID=粘贴你的-space-id
CONTENTFUL_ACCESS_TOKEN=粘贴你的-access-token
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_SITE_NAME=你的公司名称
```

3. 保存文件

---

### 第二步：安装依赖并启动项目（3分钟）

#### 2.1 打开终端

在 Cursor 中打开终端，或使用 PowerShell

#### 2.2 进入项目目录

```bash
cd D:\project\website
```

#### 2.3 安装依赖

```bash
npm install
```

**注意**：如果遇到 Node.js 环境变量问题，运行：

```powershell
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")
```

然后重新运行 `npm install`

#### 2.4 启动开发服务器

```bash
npm run dev
```

或者直接运行：

```bash
启动.bat
```

#### 2.5 访问网站

打开浏览器访问：**http://localhost:3000**

你应该能看到网站首页了！

### 第四步：配置表单提交（可选，5分钟）

#### 4.1 使用 Formspree（推荐，免费）

1. 访问：https://formspree.io/
2. 注册账号（免费）
3. 创建新表单
4. 复制表单 ID（类似：`xvgwqkny`）
5. 编辑 `app/contact/page.tsx`
6. 找到注释的 Formspree 代码（大约第 20 行）
7. 取消注释，替换 `YOUR_FORM_ID` 为你的表单 ID

```typescript
const response = await fetch('https://formspree.io/f/YOUR_FORM_ID', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(formData),
});
```

### 第五步：部署到 Vercel（免费，10分钟）

#### 5.1 准备 GitHub 仓库

1. 在 GitHub 创建新仓库（https://github.com/new）
2. 仓库名称：`website` 或 `company-website`
3. 选择 Public 或 Private
4. **不要**勾选 "Initialize with README"

#### 5.2 推送代码到 GitHub

在项目目录运行：

```bash
# 初始化 Git
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 添加远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/website.git

# 推送到 GitHub
git push -u origin master
```

#### 5.3 部署到 Vercel

1. 访问：https://vercel.com/
2. 使用 GitHub 账号登录
3. 点击 "Add New Project"
4. 导入你的 GitHub 仓库
5. 配置项目：
   - Framework Preset: **Next.js**（自动检测）
   - Root Directory: `./`（默认）
6. 添加环境变量：
   - 点击 "Environment Variables"
   - 添加 `CONTENTFUL_SPACE_ID`：填入你的值
   - 添加 `CONTENTFUL_ACCESS_TOKEN`：填入你的值
7. 点击 "Deploy"

等待 2-3 分钟，部署完成！

#### 5.4 访问你的网站

部署完成后，Vercel 会给你一个免费域名，类似：
- `your-company.vercel.app`

点击这个链接即可访问你的网站！

## 🆘 遇到问题？

### 问题 1：npm install 失败

**解决**：
```powershell
# 刷新环境变量
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")

# 重新安装
npm install
```

### 问题 2：Contentful 连接失败

**检查**：
- `.env.local` 文件中的 Space ID 和 Access Token 是否正确
- 文件是否在项目根目录
- 是否有多余的空格

### 问题 3：网站显示错误

**检查**：
- Contentful 中是否创建了内容（可选，不影响基础页面显示）
- 浏览器控制台是否有错误信息

### 问题 4：部署失败

**检查**：
- Vercel 中的环境变量是否正确配置
- GitHub 仓库是否推送成功
